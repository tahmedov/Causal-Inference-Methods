# Causal-Inference-Methods

This repository contains the code and datasets used for my bachelor's thesis:

**"Evaluating Causal Inference Methods in Natural Experiment Setup"**

## Contents

- Jupyter notebooks used for data preparation, replication, and analysis.
- Datasets used in the empirical applications.
- Results reported in the thesis.

## Data

Some datasets exceeded GitHub's file size limits and are therefore provided as compressed `.rar` archives. Extract these files before running the notebooks.

## Reproducing the Results

The analysis can be reproduced by running the Jupyter notebooks in the order presented in the thesis. Each notebook contains the necessary code for data preparation, estimation, and result generation.

## Research Objective

The thesis compares traditional Difference-in-Differences (DiD) estimators with the Cross-Moment approach proposed by Kivva et al. (2023), with a primary focus on whether both methods recover the same direction of treatment effects across several empirical applications.
